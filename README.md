Tree Maker & TnP Fitter for Muon TnP
====

# Tree Maker
* maketree_BCDEF.sh
  * script for Data periodB~F
* maketree_GH.sh
  * script for Data periodG~H
* maketree_MC.sh
  * script for MC
### Submit batch job
```source submit_maketree.sh
```
